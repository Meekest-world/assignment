The basic language of web:HTML

posted by laura jones on Monday, june 21st 2027

All mordern website and web application are built using fundamentaltechnologies. HTML,CSS and Javascript.These are languages of the web. In this post lets focus on HTML.We will learn what HTML is all about, and why you too should learn it

What ia HTML?

HTML stands for Hyper Text Markup Language. it is a markup language that web developer use to structure and describe content of a webpage (not a programming language)

HTML consist of element that describe different types of content: paragraphs, links, heading, image, video, etc Web browser understand HTML and render HTML codes as web
In HTML ,each element is made up of three parts

  1. The opening tag
  2. The closing tag
  3. The actual element

You can learn more at MDN Web Docs

Why should you learn HTML?

There are countless reason for learning the fundamental language of the Web. Here are 5 of them

  • To be able to use fundamental web development language
  • To hand-craft beautiful website instead of relying on tools like Wordpress or Wix
  • To build web application
  • To impress friends
  • To have fun 😄
